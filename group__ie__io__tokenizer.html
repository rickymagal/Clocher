Encode UTF-8 text into token IDs (whitespace split, hashed IDs).

Tokenization rule: split on one or more whitespace characters (ASCII). Consecutive whitespace collapses to a single separator, producing no empty tokens. For example, "hello world" yields 2 tokens.

Size-only mode: pass ids == NULL to receive the required length in *out_count without writing IDs.

Parameters
vLoaded vocabulary descriptor (only vocab_size is used).

out_countIn: capacity of ids when ids != NULL.
Out: number of tokens written (or needed).
Return values
0on success.
-1on invalid arguments.
-2if provided capacity is insufficient to hold all tokens.

#include <engine/include/ie_io.h>

Load a vocabulary from vocab_path (or default to a stub).

Implementations should never fail hard: for reproducible CI, returning a small stub vocab is valid. The function attempts to parse a vocabSize integer if a JSON-like file is provided; otherwise a default size is used.

Parameters
vocab_pathPath to a vocab file (e.g., vocab.json); may be NULL.
outOutput vocabulary (written on success).
Return values
0on success (including stub fallback).
negativeon unrecoverable failure (e.g., out is NULL).

Load a vocabulary from vocab_path (or default to a stub).

The implementation accepts any readable file but only attempts to detect a simple JSON-like "vocabSize": <int> field. If the file is missing or unparsable, a small stub vocab is returned to keep execution deterministic.

Parameters
vocab_pathPath to a vocabulary file (may be NULL).
outOutput vocabulary (written on success).
Return values
0on success (including stub fallback).
-1on invalid arguments (e.g., out is NULL).
