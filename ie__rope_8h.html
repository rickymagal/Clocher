Detailed Description

Rotary Position Embedding (RoPE) interfaces.

RoPE rotates interleaved pairs (2*i, 2*i+1) in each head.

This header matches the runtime usage where calls may pass only q or only k (the other pointer may be NULL), and the function returns an int status code.

RoPE scaling:

  • Scaling is implemented inside the RoPE helper (see engine/src/math/rope.c).
  • Environment variables supported by the implementation:
    • IE_ROPE_LINEAR_SCALE: positive float factor f, effective pos = pos / f
    • IE_ROPE_POS_SCALE: positive float multiplier m, effective pos = pos * m

Apply RoPE to Q and/or K for multiple heads.

Either q or k may be NULL. If both are NULL, this is a no-op returning 0.

Buffers are expected in row-major layout: q layout: [heads, head_dim] k layout: [heads, head_dim]

Apply RoPE to Q and/or K for multiple heads.

This function applies RoPE on the first head_dim elements, interpreted as interleaved pairs (even, odd): (x0, x1) -> (x0*cos - x1*sin, x0*sin + x1*cos)

The caller passes Q and K shaped as [heads, head_dim] flattened.

Parameters
qQ buffer, [heads * head_dim].
kK buffer, [heads * head_dim].
headsNumber of heads in q/k.
head_dimHead dimension (must be even for paired rotation).
posSequence position (0-based).
thetaRoPE base (commonly 10000.0f).
Returns
0 on success, negative on invalid args.
