Device-side GEMV (FP32): y = W * x (+ optional bias).

Dimensions: W is (rows x cols), x is (cols), y is (rows). Backends may copy buffers or operate in-place if memory is resident on the device.

Parameters
devDevice handle.
WRow-major FP32 weights matrix of size rows*cols.
xFP32 input vector of length cols.
yFP32 output vector of length rows.
rowsNumber of rows in W (and y).
colsNumber of columns in W (and x).
biasOptional FP32 bias vector of length rows (NULL if none).
blk_kOptional blocked-K hint (0 to ignore).
Returns
0 on success; non-zero if unimplemented or error.
