Detailed Description

Public helpers for runtime metrics (KV counters + RSS sampling).

This header complements the engine metrics exposed by ie_api.h. It provides:

  • Process-wide KV hit/miss accumulators using C11 atomics.
  • A best-effort function to retrieve the process peak RSS (in MiB).

Design:

  • We intentionally include ie_api.h (and NOT ie_metrics.h) so the project uses the canonical ie_metrics_t owned by the engine API, avoiding duplicate/contradicting typedefs.
