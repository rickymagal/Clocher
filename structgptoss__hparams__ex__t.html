Detailed Description

Extended hyperparameters typically required by a GPT-OSS forward pass.

The core engine interface uses ie_gptoss_hparams_t. A full forward pass also commonly needs:

  • RMSNorm epsilon (HF key: rms_norm_eps),
  • RoPE base theta (HF key: rope_theta),
  • optional RoPE scaling (HF object: rope_scaling),
  • whether lm_head is tied to the token embedding (tie_word_embeddings).

The ie_gptoss_hparams_t is embedded as the first member so this structure can be safely passed to APIs that expect a pointer to the core type when appropriate.
