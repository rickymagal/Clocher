11  *
12  * Design goals:
13  * - Lightweight: no libnuma dependency.
14  * - Stable ABI surface: only plain C types and a forward-declared topology handle.
15  * - Deterministic behavior: env vars define policy knobs; topology supplies facts.
16  *
17  * Environment variables (consumed by ie_numa_policy_init):
18  * - IE_NUMA_ENABLE (default: 1)
19  * - IE_NUMA_REPLICATE_HOT (default: 1)
20  * - IE_NUMA_MAX_REPLICAS (default: detected_sockets, clamped to [1, 64])
21  * - IE_NUMA_HOT_MIN_BYTES (default: 33554432 = 32 MiB)
22  */
