Detailed Description

Resolved weight handles for the full model.

This is a pure index: it does not own file mappings or dedup state. Ownership remains in gptoss_weights_index_t. Call gptoss_model_weights_free to free the per-layer array allocated by gptoss_weights_index_build_model.
