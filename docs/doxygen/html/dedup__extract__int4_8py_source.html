4 
5 Lossless dedup extractor for byte-oriented tensors (including packed INT4 weights).
6 
7 This tool creates:
8  - model.defaults.bin
9  - model.masks.bin
10  - model.exceptions.bin
11  - model.dedup.json
12

14  - A tensor is reconstructed exactly as raw bytes.
15  - The mask has 1 bit per tensor byte.
16  - Exceptions store the exact original bytes at masked indices.
17

19  1) A tensor map JSON describing where each tensor lives in the binary file.
20  Expected minimal schema:
21  {
22  "weights_bin": "model.ie.bin",
23  "tensors": [
24  { "index": 0, "name": "...", "offset": 123, "nbytes": 456 },
25  ...
26  ]
27  }
28 
29  2) A groups JSON describing dedup groups:
30  [
31  {
32  "default_index": 17,
33  "targets": [18, 19, 20]

35  ...
36  ]
37 
38 Usage:
39  python3 tools/dedup_extract_int4.py \

112 def _diff_default_target(default_bytes: bytes, target_bytes: bytes) -> Tuple[bytes, bytes]:
113  """
114  Compute (mask, exceptions) between default_bytes and target_bytes.
115 
116  The mask has 1 bit per byte; a 1 means target differs from default at that byte.
117  Exceptions are the raw target bytes at set-bit positions in ascending index order.
118  """
119  if len(default_bytes) != len(target_bytes):
120  raise ValueError("default_bytes and target_bytes must have same length")
121 
122  n = len(target_bytes)
123  mask = bytearray(_mask_bytes(n))
124  exc = bytearray()
125 
126  for i in range(n):
127  if target_bytes[i] != default_bytes[i]:
128  _bitset_set(mask, i)
129  exc.append(target_bytes[i])
130 
131  return bytes(mask), bytes(exc)
