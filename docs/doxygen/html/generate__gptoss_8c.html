Generate up to max_new tokens for a prompt string using GPT-OSS.

This function performs: 1) Load minimal hyperparameters from model_dir/config.json. 2) Encode prompt into token IDs using tok. 3) Create an inference context via ie_gptoss_infer_create(). 4) Allocate a KV cache sized from hyperparameters. 5) Run prefill over the prompt, then step+sample to generate new tokens.

The generation loop is:

  • logits = prefill(prompt)
  • repeat max_new: next = sample(logits) out_tokens[i] = next logits = step(next)

Requirements:

  • tok must be a valid tokenizer handle capable of encoding the prompt.
  • Real inference must exist in ie_gptoss_infer_prefill/step.
