Detailed Description

Reference CUDA MLP kernels (FP32).

Implements a common "SwiGLU" MLP: gate = W_gate * x + b_gate up = W_up * x + b_up act = silu(gate) * up out = W_down * act + b_down

All matrices are row-major: W rows are contiguous, W[r, c] at W[r*cols + c].

This is correctness-first; caller supplies device buffers tmp_gate/tmp_up.
