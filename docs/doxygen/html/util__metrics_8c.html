Detailed Description

Implementation of runtime metrics (KV counters + RSS sampling).

This module backs the helpers declared in util_metrics.h:

  • Process-wide accumulators for KV hits/misses using C11 atomics.
  • A best-effort function that samples the process peak RSS in MiB.

Environment:

  • Set IE_DEBUG_RSS=1 to log which source (VmHWM/VmRSS/smaps_rollup/ getrusage/mach) was used and its raw units before conversion.
