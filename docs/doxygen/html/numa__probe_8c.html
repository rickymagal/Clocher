int ie_numa_detect_nodes ( void  )

Detect the number of NUMA nodes.

Detect the number of NUMA nodes available on the system.

On Linux, reads /sys/devices/system/node/online and parses it. If the file cannot be opened, read, or yields no usable data, returns 1.

On non-Linux systems, always returns 1.

Returns
Detected NUMA node count (>= 1).

static int parse_range_list ( const char *  s)
static

Parse a NUMA node range-list and return (max_id + 1).

The input is expected to be a NUL-terminated ASCII string containing one or more comma-separated ranges. Each range is either a single integer N or a closed interval A-B where A and B are non-negative integers and A <= B. Examples:

  • "0" -> returns 1
  • "0-1" -> returns 2
  • "0-3,8-9" -> returns 10 (since max id is 9)

Parsing notes:

  • If the string cannot be fully parsed, the function returns (max_id + 1) for any successfully parsed ranges encountered before the error.
  • If no valid range is found at all, the function returns 1.
Parameters
sNUL-terminated range-list string to parse (non-NULL).
Returns
(max_id + 1) where max_id is the largest parsed id; at least 1.
