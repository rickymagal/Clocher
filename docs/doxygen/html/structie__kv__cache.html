Detailed Description

A KV cache storing K/V for one transformer layer.

The cache owns one backing allocation (a single slab) when created via ie_kv_init() or as the owner slice of ie_kv_init_layers(). Non-owner slices created by ie_kv_init_layers() point inside the owner's backing slab and must not free it.
