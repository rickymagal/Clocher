Detailed Description

Unit tests for the public engine API lifecycle.

This verifies:

  • engine creation/destruction
  • a simple generation call into a caller-provided buffer

Note: This test is executed from inside the model directory by the Makefile: ( cd models/gpt-oss-20b && ../../build/test_api ) So we pass model_dir="." to ie_engine_create().
