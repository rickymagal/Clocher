int ie_topology_init ( ie_topology_t **  topo)

Initialize and build a topology handle.

This probes the host to discover:

  • total online CPUs
  • socket id for each CPU
  • NUMA node id for each CPU A best-effort approach is used; missing information is replaced by a consistent single-socket, single-node fallback.
Parameters
[out]topoOutput pointer to receive a newly allocated handle (non-NULL).
Returns
0 on success; negative errno-like value on failure (e.g., -12 for OOM).
