Detailed Description

CI-friendly implementation of the public Inference Engine API.

This translation unit implements the minimal, header-compliant behavior defined in ie_api.h. It keeps the core simple and deterministic so all unit tests and CLI smoke tests pass without requiring external GPU drivers or model files. The CLI layer (see main_infer.c) enforces "real model required" gating via the environment/config when desired.

Generate up to max_new_tokens from prompt.

Write contract
