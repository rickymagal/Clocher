Detailed Description

Runtime CPU feature detection utilities.

This header defines a small, dependency-free API to detect ISA capabilities relevant to CPU inference kernels at runtime (e.g., selecting AVX2 GEMV).

The implementation is best-effort:

  • On x86/x86_64: uses CPUID and XGETBV to confirm both HW support and OS- enabled register state (XMM/YMM/ZMM).
  • On non-x86: reports all capabilities as false.

The detected flags are intended for:

  • Choosing kernel variants (generic vs AVX2 vs AVX-512).
  • Logging/instrumentation to confirm which fast paths are available.
