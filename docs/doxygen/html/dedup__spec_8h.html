Detailed Description

Types that describe the on-disk lossless deduplicated weights layout.

The dedup format splits weights into:

  • defaults: a baseline tensor byte stream,
  • mask: a bitset that marks which logical elements are overridden,
  • exceptions: a packed stream of overridden elements.

A tensor is reconstructed by copying defaults then patching in exceptions wherever the mask bit is 1.

The "spec" describes where each of those byte ranges live inside one or more binary files. It is intentionally lightweight and loader-agnostic.
