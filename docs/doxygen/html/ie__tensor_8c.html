Detailed Description

Minimal tensor utilities implementation (baseline).

This translation unit currently provides an out-of-line home for documentation and future extensions to the tensor utilities declared in ie_tensor.h. Vector helpers may be inline in the header for simplicity and performance; this TU exists to keep the module structure consistent and Doxygen-friendly.

Design notes

  • Keep the API surface minimal and header-inline where it makes sense.
  • Add out-of-line functions here only when they require a definition with internal linkage, platform-specific code, or non-trivial control flow.
