int main ( void )

Program entrypoint for tokenizer tests.

Steps: 1) Load vocab (file present or stub). 2) Encode with size-only query to get the required buffer length. 3) Encode into a real buffer and verify counts. 4) Decode and check formatting invariants (starts with 'T', space count).

Returns
0 on success.
