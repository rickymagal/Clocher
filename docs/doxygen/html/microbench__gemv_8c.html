Detailed Description

Standalone microbenchmark for GEMV/tanh/embed hotspots (FP32).

Build via Makefile target microbench. Runs timed loops to mimic the decode inner loop and prints plain-text metrics (time and GB/s).

Usage: ./build/microbench_gemv [H] [V] [iters] Defaults: H=256 (hidden), V=1024 (logits), iters=100
