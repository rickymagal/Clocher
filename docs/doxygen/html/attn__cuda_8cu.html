Detailed Description

Reference CUDA attention kernel (FP32) for causal self-attention.

Computes, per head: scores[t] = dot(Q[h,:], K[t,h,:]) * inv_sqrt_d out[h,:] = sum_t softmax(scores)[t] * V[t,h,:]

Layout for Q/out: [heads, head_dim] Layout for K/V cache: [seq_len, heads, head_dim] flattened with head_dim fastest: index(t,h,d) = ((t * heads + h) * head_dim + d)

This is a correctness-first implementation for a single-step decode attention.
