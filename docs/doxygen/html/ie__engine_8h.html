Detailed Description

Public engine façade (compatibility shim).

This header intentionally re-exports the stable API declared in ie_api.h without introducing new typedefs or structs. Doing so avoids duplicate definitions of:

  • ie_status_t
  • ie_metrics / ie_metrics_t
  • ie_engine_params / ie_engine_params_t
  • ie_engine_t and keeps all function prototypes single-sourced.
Note
If you previously had a standalone ie_engine.h defining its own enums/structs, remove it and use this file to prevent redefinition errors during compilation and linking.
