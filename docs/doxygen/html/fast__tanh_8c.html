Detailed Description

Fast tanh approximation (scalar and vector forms) with strict [-1, 1] clamping.

The scalar uses a cheap rational approximation for moderate |x| and early saturation for large |x|. Vector helpers call the scalar per element to keep the implementation simple and portable; SIMD paths can be added later.

Fast approximation of hyperbolic tangent with hard clamping.

Fast approximation of hyperbolic tanh with strict [-1, 1] range.

Fast scalar tanh approximation (used for fused bias+tanh paths).

For large |x| we short-circuit to +/-1.0f. For moderate |x| we use a small rational function that gives a smooth S-curve: 

tanh(x) â‰ˆ x * (27 + x^2) / (27 + 9 x^2)

Finally, the result is clamped to [-1, 1] to guarantee range correctness.
