AVX2/FMA GEMV (float path), blocked-K aware, with bias epilogue.

AVX2/FMA GEMV (float path), provided by gemv_avx2.c.

The implementation accepts either plain row-major W or rows packed as "blocked-K contiguous" (see pretranspose docs). In the latter case, each row still has cols elements but the inner loop can iterate in chunks of blk_k to promote cache locality.

Parameters
WWeights (row-major or blocked-K contiguous per row).
xInput vector (length cols).
yOutput vector (length rows).
rowsNumber of rows.
colsNumber of cols.
biasOptional bias (length rows), may be NULL.
blk_kColumn block size; 0 => treat as single block of size cols.

AVX2/FMA GEMV with per-tensor INT8 activations (two-pass: dequant then float GEMV).

AVX2/FMA GEMV with INT8 activations (per-tensor params).
