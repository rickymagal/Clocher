Detailed Description

Parsed CLI options container (benchmark/harness compatible).

This struct mirrors the soft hints consumed by ie_engine_params_t and extends them with CLI-only concerns (prompts file, batch size, etc.).

The Sparsity labels and sparsity_from_flag fields implement a simple precedence rule:

  1. If –sparsity is provided, it wins.
  2. Otherwise, IE_SPARSITY or SPARSITY environment variables are used.
  3. When both are absent, "none" is used.
