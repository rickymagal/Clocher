Detailed Description

Block-sparse GEMV kernel for single-precision weights.

This translation unit implements a reference GEMV kernel for matrices stored in ie_block_sparse_matrix_t (block-row CSR / BSR format). It is intended as a baseline, portable implementation:

  • Single-threaded by design; higher-level code is responsible for sharding work across threads if desired.
  • Numerically identical to the dense product (up to floating-point roundoff) when the block-sparse representation is exact.
  • Safe in the presence of tail blocks (partial coverage at the edges).

Perform a single-precision block-sparse matrix-vector product.

This function computes:

y = W * x (+ bias, if provided)

where W is represented in block-sparse form using ie_block_sparse_matrix_t. The input vector x has length m->cols, and the output y has length m->rows.

Tail blocks:

  • If the matrix size is not an exact multiple of the block dimensions, the implementation clips accesses to rows and cols, ensuring no out-of-bounds reads/writes occur.

Threading:

  • This function is intentionally single-threaded. Thread-level parallelism should be managed by higher-level components (e.g., a thread pool).
Parameters
mBlock-sparse matrix descriptor (must be fully initialized).
xInput vector of length m->cols (must not be NULL).
yOutput vector of length m->rows (must not be NULL).
biasOptional bias vector of length m->rows (may be NULL).
