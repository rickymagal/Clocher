Detailed Description

Replication of hot weights per socket using first-touch placement.

This unit depends on ie_topology.h for socket enumeration and thread binding. It does not define any thread-binding symbol by itself; it calls ie_topology_bind_thread_to_socket() from topology.c.

Create per-socket replicas of blob using first-touch placement.

For each socket: 1) bind to that socket via ie_topology_bind_thread_to_socket(); 2) allocate replica memory and first-touch it; 3) memcpy() the source blob; 4) hint the kernel that pages will be needed (best-effort).

Parameters
topoTopology handle (non-NULL recommended).
n_socketsSockets to build (if <= 0, derived from topo; if topo is NULL or invalid, defaults to 1).
blobSource blob (non-NULL).
blob_sizeSize of blob in bytes (> 0).
outOutput pointer to receive the replicas object (non-NULL).
Returns
0 on success; negative errno-like on failure.
