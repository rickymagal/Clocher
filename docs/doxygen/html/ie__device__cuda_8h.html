Detailed Description

C ABI wrapper for CUDA runtime functionality used by the engine.

This header intentionally exposes a small, C-callable surface so that C translation units (compiled with a C compiler) can drive CUDA work without including CUDA headers.

The implementation is provided by a .cu translation unit compiled by NVCC.
