Generic C GEMV: y = W * x (+bias), optional blocked-K layout.

The routine multiplies a row-major (or "blocked-K contiguous") matrix W by a dense vector x, accumulating into y. If blk_k is non-zero, the inner loop processes chunks of size blk_k to better match cache lines.

Parameters
WPointer to weights (row-major; each row has cols floats).
xInput vector (length cols).
yOutput vector (length rows).
rowsNumber of rows in W and y.
colsNumber of columns in W and length of x.
biasOptional bias to add per-output row (may be NULL).
blk_kColumn block size; set 0 to disable blocking.
