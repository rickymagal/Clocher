static

Build block-sparse descriptor for the 4x4 test matrix.

The layout is BSR with 2x2 blocks:

  • rows = 4, cols = 4
  • block_rows = 2, block_cols = 2
  • n_block_rows = 2, n_block_cols = 2
  • nnzb = 2 (two diagonal blocks)

Row pointers: row_ptr = [0, 1, 2] Column indices: col_idx = [0, 1] Values (two 2x2 blocks, row-major each): B0 = [1 2; 3 4] B1 = [5 6; 7 8]

Parameters
mDescriptor to populate (must not be NULL).
