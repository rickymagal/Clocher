Detailed Description

A uniform weight view describing either direct or deduplicated storage.

For IE_WVIEW_DIRECT:

  • data points to the contiguous tensor bytes
  • nbytes is the tensor byte size

For IE_WVIEW_DEDUP:

  • defaults points to the defaults tensor bytes (same dtype/shape as logical tensor)
  • mask points to a bitset over elements/nibbles
  • exceptions points to packed exception values (same dtype packing)

Mask semantics:

  • bit 0 => use defaults value
  • bit 1 => consume next value from exceptions stream and override
