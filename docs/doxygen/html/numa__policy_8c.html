Detailed Description

NUMA-aware policy initialization and helpers.

This module:

  • detects sockets via ie_topology (if available),
  • detects NUMA nodes via sysfs (ie_numa_detect_nodes),
  • reads env knobs,
  • provides decisions used by the loader and hot replication code.
