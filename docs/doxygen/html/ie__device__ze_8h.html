Detailed Description

oneAPI Level Zero backend entry points (skeleton).

This header defines a minimal C ABI for a Level Zero backend so the engine can be wired once and the implementation can evolve without touching call sites. The current implementation is a clean “unavailable” skeleton that compiles and links; it returns IE_ZE_UNAVAILABLE until we add a real kernel + runtime plumbing.

Build toggle: define IE_WITH_ZE=1 to compile/link Level Zero files.
