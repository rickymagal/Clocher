Pack two signed 4-bit integers (in [-7, +7]) into a single byte.

Values are first biased by +8 to map them to [1, 15] (0 is unused here), then stored as low/high nibbles: low for even column, high for odd column.

Parameters
q0First value (even column).
q1Second value (odd column).
Returns
