Detailed Description

Reference CPU MLP kernels (FP32).

Implements a common "SwiGLU" MLP: gate = W_gate * x + b_gate up = W_up * x + b_up act = silu(gate) * up out = W_down * act + b_down

All GEMVs use ie_gemv_f32() so the installed kernel backend is honored. This implementation is allocation-free: caller provides temporaries.
